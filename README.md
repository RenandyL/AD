<h1>Active Directory</h1>
This project outlines the process of unlocking a user account and password reset in Active Directory.<br />



<h2>Environments and Technologies Used</h2>

- Microsoft Azure (Virtual Machines/Compute)
- Remote Desktop
- Active Directory Domain Services
- osTicket


<h2>Operating Systems Used </h2>

- Windows Server 2022
- Windows 10 (21H2)

<h2>High-Level Deployment and Configuration Steps</h2>

- Log in to Azure, create a virtual machine
- Install Active Directory within the VM
- Receive user ticket for password reset or unlocking account in osTicket
